## 📌 Addendum (2026-04-20, release-prep on f676d0e)

Both rc3 blockers landed before the release tag:

- **#225 Pi footer (V7) — FIXED** in PR #339 (commit `efa60a0`). Live-verified on `@untether_dev_bot`. Issue closed.
